# Color

Color system is based on the HOT branding document and are inspired in both WebAwesome and Tailwind.

You can access any color using CSS variables:

`var(--hot-color-{name}-{number})`

For example:

```css
.dangerMessage {
  color: var(--hot-color-danger-700);
  background: var(--hot-color-danger-50);
}
```
