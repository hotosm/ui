# Typography

You can use typography CSS variables for family, size, weight, line-height and letter-spacing,
for example:

```css
.title {
  font-size: var(--hot-font-size-2x-large);
  letter-spacing: var(--hot-letter-spacing-dense);
  font-weight: var(--hot-font-weight-light);
  line-height:  var(--hot-line-height-normal);
}
```

## Font family

You have to include the following web fonts if you want to use them:

* [Archivo](https://fonts.google.com/specimen/Archivo) (font-sans)
* [Barlow Condensed](https://fonts.google.com/specimen/Barlow+Condensed) (font-sans-variant)
* [Space Mono](https://fonts.google.com/specimen/Space+Mono) (font-mono)
