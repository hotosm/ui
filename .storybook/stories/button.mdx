# Button Component

The `hot-button` component is a customizable button built on top of WebAwesome's button component with HOT theming.

## Features

- Multiple variants (primary, secondary, outline, success, warning, danger)
- Different sizes (small, medium, large)
- Icon support with configurable positioning
- Loading and disabled states
- Custom HOT-themed variants
- Full WebAwesome integration

## Basic Usage

```html
<hot-button variant="primary">Click me!</hot-button>
```

## API Reference

### Properties

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `variant` | `'primary' \| 'secondary' \| 'outline' \| 'success' \| 'warning' \| 'danger'` | `'primary'` | The button variant to use |
| `size` | `'small' \| 'medium' \| 'large'` | `'medium'` | The button size |
| `appearance` | `'default' \| 'icon'` | `'default'` | The button appearance |
| `disabled` | `boolean` | `false` | Whether the button is disabled |
| `loading` | `boolean` | `false` | Whether the button is in loading state |
| `icon` | `string` | `''` | The icon name to display |
| `iconPosition` | `'start' \| 'end'` | `'start'` | The icon position relative to the text |
| `type` | `'button' \| 'submit' \| 'reset'` | `'button'` | The button type |
| `hotColor` | `'red' \| 'yellow' \| 'blue' \| 'cyan' \| 'rose' \| 'gray' \| ''` | `''` | The HOT color variant to use |

## React Usage

For React applications, use the `WaHotButton` wrapper:

```jsx
import { WaHotButton } from '@hotosm/ui';

function MyComponent() {
  return (
    <WaHotButton variant="primary" size="medium" onClick={() => console.log('clicked')}>
      Click me!
    </WaHotButton>
  );
}
```
